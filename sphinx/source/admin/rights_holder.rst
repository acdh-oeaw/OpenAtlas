Rights holder
=============

.. toctree::

Rights holders are used to define the creator or license holder of a file.
In the overview, you can see all rights holders and their type and description.
If **Actor** is written in the type column, it means that this rights holder
has no type and is most likely from older data. If this entry is updated, it
needs to be changed to either **Person** or **Group**.
When viewing a rights holder, you can see all files linked to it.

Form fields
-----------

* **Name** - the name of the person or group
* **Type** - can either be **Person** or **Group** (similar to CIDOC
  entities :cidoc_entity:`E21 Person<e21-person>` and
  :cidoc_entity:`E74 Group<e74-group>`). Note that this distinction is only
  used for internal organization and is not part of the CIDOC CRM model.
* **Info** - a description or additional information about the person or group
